Промышленный льдогенератор для нужд цеха покраски

Сейчас этот проект просматривает 4 человека

Площадь обьекта

245 500 м2

01. О проекте

Компанией «Новая Линия» выполнена разработка проекта и поставка оборудования для системы получения чешуйчатого льда для технологического процесса покрасочного цеха автомобильного завода ООО «Хендэ мотор мануфактуринг рус». Система производительностью 9 тонн льда в сутки реализована с применением промышленного льдогенетатора производства Maja (Германия).

Холодильная система для намораживания льда выполнена на компрессорном агрегате на базе полугерметичного поршневого компрессора Bitzer (Германия) с водяным конденсатором.
